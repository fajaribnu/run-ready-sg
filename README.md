# 🏃‍♂️ RunReady SG

> **Singapore's first location-aware, weather-safe outdoor activity planner.**

RunReady SG is a Progressive Web App (PWA) designed to keep runners, cyclists, and Physical Education (PE) teachers safe from unpredictable tropical weather. By aggregating real-time heat stress (WBGT), lightning alerts, and localized rain forecasts, RunReady SG provides instant safety recommendations and dynamic emergency routing to the nearest shelter.

## ✨ Core Features

* 🚦 **Safe-to-Go Dashboard:** Instant color-coded safety recommendations (Proceed / Modify / Move Indoors) based on live meteorological data at your specific location.
* ⚡ **Find Shelter Now:** A prominent mid-workout panic button that uses live GPS to instantly route you to the nearest HDB or Civil Defence shelter.
* 🌡️ **Personalized WBGT Coach:** Translates raw Wet Bulb Globe Temperature data into actionable health advice tailored to your acclimatization profile.
* 🗺️ **Safe Run Generator:** Plans routes that avoid incoming rain cells and maximize coverage under structural walkways or tree canopies.

## 🛠️ Tech Stack

**Frontend (PWA)**
* React.js (Vite)
* Tailwind CSS
* React-Leaflet (Mapping)

**Backend & Cloud Infrastructure**
* Python FastAPI
* PostgreSQL with PostGIS (Spatial routing)
* AWS EC2 / Lightsail (Hosting & Cron Jobs)

**Data Pipelines**
* **National Environment Agency (NEA) APIs:** Real-time WBGT, 2-Hour Forecast, and Lightning Observations.
* **OneMap API:** Pedestrian routing engine.
* **NParks & OpenStreetMap:** Park geometries and covered walkway nodes.
